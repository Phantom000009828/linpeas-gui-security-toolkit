# LinPEAS GUI Security Toolkit

A GUI-based Linux Privilege Escalation Scanner inspired by LinPEAS.

## Features

- System Information Enumeration
- SUID Binary Detection
- Weak Permission Scanning
- Cron Job Vulnerability Detection
- Kernel Vulnerability Check
- Risk Scoring Dashboard
- Security Report Generation

## Technologies

- Python
- CustomTkinter
- Linux Security Enumeration

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the scanner:

python main.py
